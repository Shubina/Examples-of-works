Программы обучения

Базовый курс для визажистов

  • Стоимость: 6500 грн/21 занятие
  • Количество человек в группе: 5-6
  • Преподаватель: Елена Крыгина
  • Продолжительность занятия: 3-4 часа
  • Косметика для работы и модели предоставляются
  • Выдается диплом об окончании курса

Курс “Макияж для себя”

  • Стомость: 1500 грн / 3 занятия
  • Индивидуальные занятия
  • Преподаватель: Елена Крыгина; Юлия Иванова
  • Продолжительность занятия: 1.5 часа
  • Научим вас делать макияж самостоятельно для повседневной жизни и для выхода в свет

Курс “Body-art”

  • Стоимость 3500 грн/12 занятий
  • Количество человек в группе: 4-5
  • Преподаватель: Юлия Иванова
  • Продолжительность занятия: 4-5 часов
  • Косметика для работы и модели предоставляются
  • Выдается диплом об окончании курса
